## Project Overview
This final project was completed to develop practical T-SQL skills through SQLNOIR mystery-solving tasks and business-oriented SQL analysis. The project includes individual SQL mysteries and pair session style business problem-solving using the Northwinds2024Student database.

The work demonstrates SQL querying, joins, aggregations, common table expressions (CTEs), window functions, user-defined functions, recursive CTEs, and stored procedures.

---

## Project Goals
- Strengthen practical T-SQL skills
- Use joins, functions, aggregations, subqueries, and CTEs
- Explore more advanced SQL features such as window functions and recursive CTEs
- Solve business-style SQL problems
- Present results in a structured and well-documented format
